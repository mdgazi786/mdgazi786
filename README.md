## Featured projects

**[PineScript AI (pinedev)](https://github.com/mdgazi786/pinedev)**
A self hosted, AI powered assistant for generating, validating, and optimizing TradingView PineScript v6. Built with SvelteKit 2 and Svelte 5 in TypeScript. Streaming chat interface, live CodeMirror editor, BM25 RAG grounding over Pine docs, a multi stage validation pipeline (static rules, transpiler check, AI review, auto correction), and multi provider LLM support (Anthropic, OpenRouter, MiniMax, Ollama) with encrypted BYOK key storage.

**[terminal-cyber-security-agent](https://github.com/mdgazi786/terminal-cyber-security-agent)**
A terminal based cybersecurity agent, written in TypeScript.

**[Svelte Quiz App](https://github.com/mdgazi786/Svelte-Quiz-App-timer)**
A quiz app built in Svelte with a timer.

**[simple_http_rust_server](https://github.com/mdgazi786/simple_http_rust_server)** and **[number_guessing_game](https://github.com/mdgazi786/number_guessing_game)**
